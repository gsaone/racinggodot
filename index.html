Project Features

Custom Physics Model

Car implementation is based on RigidBody for collision calculations and RayCast rays for suspension system.

AI Opponents

Implemented AI-controlled opponents that follow the track using Path curves for challenging gameplay.

Dynamic Camera System

Multiple view options (first-person, third-person, static camera) that can be switched with a single button press.

Immersive Audio

Engine sounds with RPM-based pitch changes and realistic tire skidding effects for enhanced driving experience.
