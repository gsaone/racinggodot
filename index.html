RacingGodot

Custom arcade car physics implementation for Godot Engine without using the built-in VehicleBody class

Built with RigidBody & RayCast

Custom physics implementation without using the built-in VehicleBody class. Car physics is based on RigidBody for collisions and RayCast for suspension.

Demo Video

Features

  • P Custom Physics Model

    Car implementation based on RigidBody for collision calculations and RayCast rays for suspension system. Full control without relying on VehicleBody.

  • AI AI Opponents

    Implemented AI-controlled opponents that follow the track using Path curves. Provides challenging and dynamic gameplay.

  • C Dynamic Camera System

    Multiple view options (first-person, third-person, static) that can be switched instantly. Various camera types for different perspectives.

  • S Immersive Audio

    Engine sounds change based on RPM, tire skidding effects, and immersive audio design for enhanced driving experience.
