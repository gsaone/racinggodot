RacingGodot

Final project featuring custom arcade car physics implementation for Godot Game Engine

Custom physics implementation without using the built-in VehicleBody class. Car physics is based on RigidBody for collisions and RayCast for suspension.

Project Demo Video

Project Features

Custom Physics Model

Car implementation is based on RigidBody for collision calculations and RayCast rays for suspension system. Full control and flexibility in tuning car behavior without relying on the built-in VehicleBody class.

AI Opponents

Implemented AI-controlled opponents that follow the track using Path curves. Racer AI navigates the road according to predefined routes, providing challenging gameplay.

Dynamic Camera System

Multiple view options (first-person, third-person, static camera) that can be switched with a single button press. Several camera types implemented for varied gameplay perspectives.

Immersive Audio

Engine sounds — pitch and intensity change based on RPM and player actions.
Tire sounds — skidding and drifting sound effects implemented for enhanced driving feel and speed perception.

Game Controls

Use these keyboard controls to play the game in the web demo or downloaded version:

↑

Forward / Accelerate

Press to accelerate the car forward

←

Turn Left

Steer the car to the left

→

Turn Right

Steer the car to the right

↓

Reverse / Brake

Slow down or move backward

SPACE

Hand Brake

Engage hand brake for drifting

A

Add Opponents

Add AI opponents to the race

D

Remove All Opponents

Clear all AI opponents from the track

B

Look Back

Temporary rear view camera

V

Change View

Cycle through different camera perspectives

Tip: These controls are fully customizable in the source code. You can modify key bindings or add support for gamepad/joystick input.
